# Student Grade Tracker - Part 1

In this assignment, I created a simple command-line Student Grade Tracker using basic Python concepts like loops, conditions, lists, and strings.

## Task 1 - Data Cleaning
In this task, I cleaned the raw student data. The names had extra spaces and mixed letter cases, so I removed the extra spaces and converted the names into title case. I converted roll numbers from string to integer. I also converted the marks from a comma-separated string into a list of integers. Then I checked whether each student name is valid (only alphabets). After cleaning the data, I printed a formatted student profile for each student. I also printed the name of the student with roll number 103 in uppercase and lowercase.

## Task 2 - Marks Analysis
In this task, I used loops and conditions to analyse student marks. I printed each subject with marks and grade. Then I calculated total marks and average marks. I also found the highest and lowest scoring subjects. After that, I created a marks entry system using a while loop where the user can add new subjects and marks. I also handled invalid inputs so that the program does not crash.

## Task 3 - Class Performance Summary
In this task, I calculated the average marks of each student and assigned pass or fail based on the average. I printed a formatted class performance table. I also counted how many students passed and how many failed. Then I found the class topper and calculated the overall class average.

## Task 4 - String Manipulation Utility
In this task, I performed different string operations on a paragraph. First, I removed extra spaces using strip(). Then I converted the text into title case. I counted how many times the word "python" appears in the paragraph. After that, I replaced the word "python" with "Python 🐍". Finally, I split the paragraph into sentences and printed each sentence with numbering.

This assignment helped me understand loops, conditions, lists, and string functions in Python.
